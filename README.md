# AI Prompt Sharing & Marketplace Platform (Server)

## Purpose

This server provides REST APIs for the AI Prompt Sharing & Marketplace Platform. It handles authentication, authorization, prompt management, bookmarks, reviews, premium subscriptions, payments, analytics, and administrative operations using Express.js and MongoDB.

## Live URL

https://ai-prompt-server-six.vercel.app

## Key Features

- RESTful API
- JWT Authentication
- Role-Based Authorization
- MongoDB Database
- Prompt CRUD Operations
- Bookmark System
- Review & Rating APIs
- Prompt Reporting
- Stripe Payment Integration
- Premium User Management
- Admin Moderation
- Backend Search, Filter & Sorting
- Pagination
- MongoDB Aggregation
- Secure Environment Variables
